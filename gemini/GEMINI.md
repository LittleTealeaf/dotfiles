# Gemini & Antigravity Configuration

Welcome to the configuration directory for your AI developer assistants (**Gemini / Antigravity**).

This folder is designed to store and manage configurations, specialized behaviors, and custom skills for your AI coding companions.

## Directory Structure

*   **`skills/`**: Custom scripts, tools, or prompt instructions defining specialized tasks your AI assistants can perform.
*   **`GEMINI.md`**: General documentation, workflow guidelines, and reference notes.

## Project & Directory Organization

Always organize files and workspaces according to these rules:

1. **Development Projects**: Put active software development projects, repositories being authored, and personal coding workspaces under `~/dev/` (e.g., `~/dev/my-project/`).
2. **Third-Party & Non-Development Repositories**: Put cloned external repositories, third-party source code, and tools downloaded/built from git under `~/git/` (e.g., `~/git/some-tool-to-build/`).
3. **Miscellaneous & Temporary Files**: Put throwaway scripts, quick drafts, temporary outputs, and miscellaneous logs/tests under `~/tmp/`.
